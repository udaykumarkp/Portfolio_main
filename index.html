Projects

Precision Laser Alignment and Positioning System

Developed an advanced automatic positioning system for laser focusing lenses, utilizing a voice coil motor for precision control. The system integrates accelerometers and input/output interfaces to achieve accurate height adjustments and motion stability. Leveraged EtherCAT and CAN protocols for real-time communication, ensuring seamless control and reliable data exchange in dynamic environments.

PyQt-based GUI for Touchscreen with EtherCAT Communication

Developed an interactive, user-friendly GUI using PyQt for a touchscreen interface integrated with Toradex and Raspberry Pi platforms. The project enables real-time EtherCAT communication for controlling devices and visualizing data, designed for industrial automation systems.

  • User-friendly GUI: Intuitive touch interface built with PyQt.
  • Real-time Communication: EtherCAT for high-speed, deterministic data exchange.
  • Custom Touchscreen Controls: Optimized for Toradex and Raspberry Pi hardware.
  • Cross-platform: Runs seamlessly on multiple devices.

Wi-Fi Controlled LED

Developed a system where an LED connected to an STM32 microcontroller is controlled via a mobile device using the ESP8266 Wi-Fi module. The system allows remote control of the LED over a Wi-Fi network by sending HTTP requests from a mobile browser or app.

  • STM32 microcontroller controls the LED.
  • ESP8266 module connects to a Wi-Fi network and acts as a web server.
  • Mobile device sends HTTP requests to control the LED on/off state.
  • Implemented using Arduino IDE for ESP8266 programming and STM32CubeIDE for STM32 programming.
